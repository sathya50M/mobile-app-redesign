#DESCRIPTION:

This project is a modified version of the Instagram app, created as part of an app customization task. In the original version, the chat section contained a search box that allowed users to search through messages. However, multiple steps were required to access and use it. To simplify the process and improve usability, the search box feature in the chat section has been disabled in this version.

Features:

Based on the Instagram app UI.

Disabled search box in the chat area for easier navigation.

Focus on improving user experience through simplified interaction.

purpose:
This project demonstrates how small changes in app features can impact usability and interface design.
